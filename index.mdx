# SkinTrack

SkinTrack is a private operational app for cosmetology practices.

It ships with:

- secure staff authentication with Supabase Auth
- patient records with ownership-based access
- clinical sessions with draft mode and autosave
- superadmin-controlled invitations and user management

## Stack

- Next.js 15 App Router
- Supabase Auth + PostgreSQL + RLS
- Tailwind CSS 4
- Motion
- Zustand
- Zod
- Vitest

## Current modules

- Auth
- Profile
- Users
- Patients
- Clinical Sessions
